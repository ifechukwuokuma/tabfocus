Installation

Free. No store needed.

TabFocus isn't on the Chrome Web Store — just download and load it manually. Here's how.

1
Download the extension

Click the download button and save the ZIP file anywhere on your computer.

2
Unzip the folder

Extract the ZIP file. You should have a folder called tabfocus.

3
Open Chrome Extensions

In Chrome, go to chrome://extensions/ and enable Developer mode using the toggle in the top right.

4
Load the folder

Click Load unpacked and select the tabfocus folder you just unzipped.

5
Done!

TabFocus appears in your toolbar. Click the icon to open the popup and make sure it says Active.
